# class `shogun::CLatentSVM` {#classshogun_1_1CLatentSVM}

```
class shogun::CLatentSVM
  : public CLinearLatentMachine
```

LatentSVM class Latent SVM implementation based on [1]. For optimization this implementation uses SVMOcas.

User must provide a her own [CLatentModel](#classshogun_1_1CLatentModel) which implements the PSI(x_i,h_i) function for the given problem.

[1] P. F. Felzenszwalb, R. B. Girshick, D. McAllester, and D. Ramanan, "Object detection with discriminatively trained part-based models," Pattern Analysis and Machine Intelligence, IEEE Transactions on, vol. 32, no. 9, pp. 1627-1645, 2010.
