# class `shogun::FirstOrderStochasticMinimizer` {#classshogun_1_1FirstOrderStochasticMinimizer}

```
class shogun::FirstOrderStochasticMinimizer
  : public FirstOrderMinimizer
```

The base class for stochastic first-order gradient-based minimizers.

This class gives the interface of these stochastic minimizers.

A stochastic minimizer is used to minimize a cost function $f(w)$ which can be written as a (finite) sum of differentiable functions, $f_i(w)$. (eg, [FirstOrderStochasticCostFunction](#classshogun_1_1FirstOrderStochasticCostFunction)) For example, 
$$
f(w)=\sum_i{ f_i(w) }
$$
 Note that we call these differentiable functions $f_i(w)$ as sample functions.

This kind of minimizers will find optimal target variables based on gradient information wrt target variables. [FirstOrderStochasticMinimizer](#classshogun_1_1FirstOrderStochasticMinimizer) uses a sample gradient (eg, [FirstOrderStochasticCostFunction::get_gradient()](#classshogun_1_1FirstOrderStochasticCostFunction_1a2964fe0d88ef68c45dcc1f999f7187c6) ) $\frac{\partial f_i(w) }{\partial w}$ to find optimal target variables, where the index $i$ is generated by some distribution (eg, [FirstOrderStochasticCostFunction::next_sample()](#classshogun_1_1FirstOrderStochasticCostFunction_1afe56e2ec0f03dd3f22294cdbc4ada815) ).

Note that [FirstOrderMinimizer](#classshogun_1_1FirstOrderMinimizer) uses the exact gradient, (eg, [FirstOrderCostFunction::get_gradient()](#classshogun_1_1FirstOrderCostFunction_1a2964fe0d88ef68c45dcc1f999f7187c6) ), $\frac{\partial f(w) }{\partial w}$.

For example, least sqaures cost function 
$$
f(w)=\sum_i{ (y_i-w^T x_i)^2 }
$$
 If we let $f_i(w)=(y_i-w^T x_i)^2 $, $f(w)$ can be written as $f(w)=\sum_i{ f_i(w) }$. Note that $f_i(w)$ is a sample function for the i-th sample, $(x_i,y_i)$.
